### Описание проекта:

Разработка автотестов для проверки создания наборов продуктов в Яндекс Прилавке

:honeybee: Для запуска тестов должны быть установлены пакеты pytest и requests.
:tulip: Запуск всех тестов выполняется командой pytest.
